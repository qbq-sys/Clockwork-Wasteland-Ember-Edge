# 《发条荒野：余烬边境》第一版技能资产清单

版本：V1  
用途：把《第一版职业角色设计方案》拆成可执行的内容制作清单，指导第一轮 `SkillData`、`BuffData`、专精被动与角色挂载。  
适用范围：当前四个现有角色原型。  

---

## 1. 使用说明

这份文档不是讲理念，而是讲“第一轮要做哪些资产”。

每个角色拆成四块：

1. 角色定义资产
2. 技能资产
3. Buff / Debuff 资产
4. 专精与被动挂接点

文档默认规则：

- 每个角色先落 4 个初始技能
- 2 个专精方向先定名字和核心被动，不急着一次做满全部专精技能
- Buff 先做通用基础款，技能挂接尽量复用
- 第一轮先跑通职业身份，不追求技能池庞大

---

## 2. 通用基础资产清单

这些资产建议先于四职业单独创建。

### 2.1 通用系统技能

#### `skill_swap_position`
- 名称：换位
- 类型：控制 / 位移
- 目标：单友
- 用途：战斗内交换站位

#### `skill_pass_turn`
- 名称：跳过回合
- 类型：控制
- 目标：自己
- 用途：当回合无更优操作时结束行动

### 2.2 第一版通用 Buff / Debuff

#### `buff_bleed`
- 名称：流血
- 用途：持续掉血
- 颜色：红

#### `buff_poison`
- 名称：中毒
- 用途：持续掉血
- 颜色：绿

#### `buff_shadow_rot`
- 名称：暗蚀
- 用途：巫术异常 / 持续伤害
- 颜色：暗紫

#### `buff_marked`
- 名称：锁定
- 用途：被追击 / 被增伤 / 被联动
- 颜色：橙黄

#### `buff_slow`
- 名称：减速
- 用途：降低速度或出手质量
- 颜色：灰蓝

#### `buff_guard`
- 名称：护卫
- 用途：承受队友伤害
- 颜色：土黄

#### `buff_damage_down`
- 名称：压制
- 用途：降低目标输出
- 颜色：暗棕

#### `buff_evasion_up`
- 名称：规避提升
- 用途：短时提高生存
- 颜色：灰白

#### `buff_regen`
- 名称：持续恢复
- 用途：每回合恢复生命
- 颜色：淡绿

#### `buff_execution_window`
- 名称：断首
- 用途：Executioner 击杀后强化
- 颜色：血红

### 2.3 第一版专精被动实现建议

第一轮建议不要立刻做复杂天赋树。  
专精核心被动可以先用两种方式之一承载：

1. 写进 `CombatantDefinition` 的专精判断逻辑
2. 做成“隐藏职业被动”并在战斗逻辑里按专精触发

第一轮目标是功能成立，不强求编辑器里已经有完整被动管理器。

对应角色：和服武士刀狼人

- 初始站位偏好：2 / 3

### 3.2 初始技能资产

#### `skill_exec_shadow_slash`
- 名称：影切
- 类型：伤害
- 目标：单敌
- 消耗：1
- 冷却：0
- 站位要求：2 / 3
- 联动：
  - 对负面状态目标额外增伤

#### `skill_exec_severing_cut`
- 名称：裂脉斩
- 类型：伤害 / 异常
- 目标：单敌
- 消耗：2
- 冷却：1
- 站位要求：1 / 2 / 3
- 挂接 Buff：
  - `buff_bleed`

  - 自建流血条件

#### `skill_exec_crescent_lunge`
- 名称：残月突进
- 类型：伤害 / 位移
- 目标：单敌（偏后排）
- 消耗：2
- 冷却：2
- 站位要求：2 / 3
- 额外逻辑：
  - 自身前移 1 格

#### `skill_exec_wild_hunt`
- 名称：野性追猎
- 类型：伤害 / 收割
- 目标：单敌
- 消耗：3
- 冷却：2
- 站位要求：2 / 3
- 额外逻辑：
  - 对残血目标高额伤害
  - 击杀回资源

### 3.3 初始职业被动

#### `passive_exec_hunter_instinct`
- 名称：猎首本能
- 效果：
  - 残血增伤
  - 对异常目标攻击时回资源

### 3.4 专精清单

#### 专精A：`spec_exec_reaper`
- 中文名：血月收割者
- 核心被动：`passive_exec_reaper_echo`
  - 中文名：断首回响
  - 效果：
    - 击杀后获得 `buff_execution_window`

#### 专精B：`spec_exec_vanguard`
- 中文名：边境先锋刃
- 核心被动：`passive_exec_vanguard_breakline`
  - 中文名：破阵猎手
  - 效果：
    - 攻击后排增伤
    - 攻击异常后排时回资源

### 3.5 第一轮最小落地目标

必须先做：

- `skill_exec_shadow_slash`
- `skill_exec_severing_cut`
- `skill_exec_wild_hunt`
- `passive_exec_hunter_instinct`
- `buff_bleed`

## 8. 第一轮落地顺序建议

### 阶段1：Bulwark + Executioner

目的：

- 先立住前排压制与收割关系
- 验证位置战术有没有成立

### 阶段2：Physician

目的：

- 加入治疗、净化、续航
- 验证战斗是否从纯对打变成“压力管理”

### 阶段3：Artificer

目的：

- 加入标记、异常、控场与资源回转
- 把构筑联动真正拉起来
